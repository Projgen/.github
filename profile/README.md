## What is Projgen?

`npx projgen`

Projgen is an interactive CLI that scaffolds new projects in seconds. Pick your tech stack, add optional integrations, and start coding — no manual config required.

- **Any stack** — React, Next.js, Vite, Node, and more
- **Customizable recipes** — define your own stack presets as JSON/YAML
- **Zero opinion lock-in** — you own every generated file
- **TypeScript-first** — the CLI and all recipes are fully typed
